Only this README is tracked on GitHub. Everything else you drop into this folder stays local (see .gitignore).

Use it for:
- Scratch notes, one-off scripts, or experiments that are not part of the diet/fridge contribution
- Extra .env backups (never commit real secrets)
- Large personal datasets you do not want in the shared repo

Your teammate's branch should only include the diet + fridge changes documented in docs/BRANCH_PUSH_CHECKLIST.md.

You can copy this entire folder somewhere safe, then delete it from the project before opening a PR if you prefer a clean tree.
